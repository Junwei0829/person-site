# Personal Website

This is a static personal website that can be deployed directly to GitHub Pages.

## What to Edit

Replace these items in `index.html`:

- `Your Name`
- `you@example.com`
- `https://github.com/yourname`
- `https://www.linkedin.com/in/yourname`
- Project, experience, and skill descriptions

The hero image is located at `assets/hero-workspace.png`.
You can replace it with your own photo or project image.

## Deploy to GitHub Pages

1. Create a new GitHub repository, for example `yourname.github.io`.
2. Upload the files in this folder to the repository root.
3. Open the repository's `Settings` -> `Pages`.
4. Set Source to `Deploy from a branch`.
5. Set Branch to `main`, select `/root`, and save.
6. Wait a minute or two, then visit `https://yourname.github.io`.

If the repository is not `yourname.github.io`, for example this repository is named `personal-website`,
the website URL is usually:

```text
https://yourname.github.io/personal-website/
```
